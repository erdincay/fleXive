Adds a TAB to the [fleXive] backend tab-bar. This is done by creating a invisible jsf command link, and then copying and usingthe the onclick even of this command link on the tab bar. Parameters: label - the tab label labelKey - the message key for the tab label (overrides label) bean - JSF bean for invoking the action action - name of the action to be invoked lockscreen - if false, the screen will not be locked when clicking on this tab immediate - if clicking on the tab should skip JSF validation and update model phases rendered - if the tab should be rendered #{label}

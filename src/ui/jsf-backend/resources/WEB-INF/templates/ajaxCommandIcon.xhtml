Backend a4j command icon. Parameters: id - element id reRender - the region to re-render title - [optional] the button title (used by browsers to display tooltip on mouseover) titleKey - [optional] message key for the button title (overrides title when specified). if not specified and labelKey is set, it defaults to #{labelKey}.title bean - the bean that the action belongs to actionr - the action to call immediate - [optional] the jsf immediate flag onclick - [optional] the onClick event of the action link icon - [optional] the name of the icon to be displayed. the icon name is mapped to the url "pub/images/commandButton/${icon}.png" iconUrl - [optional] the url (aboslute, including the context path) of the icon to be displayed (i.e. /flexive/images/myIcon.png) eventsQueue - ajax events queue rendered - [optional] flag to supress rendering of the whole component iconDisabled - [optional] flag to disable the component oncomplete - [optional] the oncomplete event of the command link [id of ajaxCommandIcon not set]

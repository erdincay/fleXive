A YUI-based color picker shown in a popup dialog. The color value will be written to the given form element (an input text or hidden input). The color-picker can be nested in a JSF input element (e.g. h:inputText). Parameters: inputId - ID of the input element where the picked color value will be stored. If not set, the parent component ID will be used.
